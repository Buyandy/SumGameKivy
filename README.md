# SumGameKivy

Проект создан параллельно с обучением самой мощной библотеки Kivy. Приложение представляет из себя самую обычную игру про сложение чисел, тоесть ты просто выбираешь какой правильный ответ выйдет при сложении двух чисел.

Если кто хочет сыграть или рыпатся в самом проекте то вот гайд по сборке. (сам проект не компилировался на телефонах потому что возникали проблемы скачивании нужных ресурсов для компиляции, из-за блокировок)

## Скачивание проекта

```sh
git clone https://github.com/Buyandy/SumGameKivy.git
cd SumGameKivy
```

## Сборка и запуск

Сам проект создан через канцелярский нож `uv`. Поэтому проект рекомендуется запускать через этот пакет.

```sh
uv run main.py # Сам соберет зависимости и запустит проект в космос
```

### Если нету uv

```sh
pip install uv
```
